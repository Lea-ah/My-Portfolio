Hi,  I'm Lea
a former hospitality manager reconverted to Tech !

WHY TECH?

I've always seen hospitality as a problem-solving game : how can I provide the best experience while being fast, organised and able to communicate efficiently with my co-workers & customers in a short amount of time?
During the pandemic as a Manager I really learnt how to use technology to my advantage (especially being short-staffed):
- Improving the user experience on the website to avoid recurring phone calls (using WIX)
- Making the Square POS system as easy as possible to use for the new team members
- Online tools such as Asana, Deputy to communicate efficiently with my team

That's when my interest for UI / UX design took shape
So I checked what was needed to be a Product Designer and some coding knowledge seemed in demand
It was time to learn coding !

CODING JOURNEY

April 2022: I joined 'Le Wagon' Full Stack bootcamp
( Number 1 acclaimed bootcamp worldwide ! )
And My oh My, what an exciting journey !

I have learned to:
- Code in Back-end & Front-End Languages ( but my heart goes to Front-end )
- Work on projects as a Team using Trello & Zoom for better communication
- Pair-programing
- Prioritizing to deliver a good UX experience
- Be uncomfortable, feeling dumb but not giving up

Understanding coding opened new doors in what was possible in Design too & Understanding the technical terms
